# Habit Tracker

A small habit tracking app built with React, TypeScript, Vite, and Tailwind CSS.

This project was recreated from the tutorial in [this YouTube video](https://www.youtube.com/watch?v=9aTRnV6g0eQ) as practice for using Tailwind.

## Features

- Create new habits
- View and manage your habit list
- Persist data in local storage
- Styled with Tailwind utilities and `tailwind-merge`

## Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Available scripts:

- `npm run dev` - start the Vite dev server
- `npm run build` - type-check and build for production
- `npm run lint` - run ESLint
- `npm run preview` - preview the production build
